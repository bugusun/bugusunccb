# 更新记录

## 2026-04-17

- 新增火箭炮：低射速、高伤害、范围爆炸，并带击飞、尾焰与爆炸演出
- 火箭炮接入专属升级“广域装药”，可提升爆炸半径，并在商店与升级池按武器过滤
- 每个首次进入的战斗 / 精英 / 首领房都会给短暂缓冲，并把怪物刷新得更远，避免突脸
- 略微缩小 `toxic_bloater` 与毒性障碍的毒雾范围
- 继续压缩左上 HUD，统一升级 / 奖励 / 补给卡片布局，并收短标题页与交互文案
- 降低 3x3 迷宫房频率，新增 1x2、1x3 与中央环形墙房模板


## 2026-04-16

- 修复 `README.md` 与 `CHANGELOG.md` 的中文乱码问题
- 新增 `game/content.py`，集中管理升级、角色、武器、补给与标题页文案
- 新增 `game/balance.py`，集中管理敌人数值缩放、商店涨价与奖励掉落公式
- 降低大部分升级的单次提升幅度，使成长曲线更平滑
- 新增升级选项：降低敌方子弹速度、提高晶片获取、子弹反弹、扩大脉冲范围
- 商店物品改为随楼层与房间难度涨价，后期资源决策更重要
- 敌人的生命、接触伤害与远程伤害会随楼层继续成长
- 子弹新增一次反弹能力，并在 HUD 中显示关键成长状态
- 升级抽取会规避部分已到上限的选项，减少无效选择
- 调整补给房、晶片掉落与部分即时道具数值，平衡中后期节奏
- Boss 房出口改为二次确认进入下一层，并加入短转场
- 爆炸桶伤害提高 50%，机枪怪（`shooter`）射击频率降低
- 普通枪反射上限保持 1 次；激光枪可通过升级叠加反射至 3 次，达到上限后不再出现该升级
- Boss 新增近距离撼地与中距离震荡齐射攻击
- 新增 `game/navigation.py`，将怪物绕障碍寻路抽离为独立导航模块
- 交付前补做 `py_compile` 与 dummy SDL smoke test，确认主要流程可运行
- `elite` 重做为高攻低防的 burst 精英单位，修复其异常高频射击问题
- 新增霰弹枪武器、霰弹枪独有升级“加长枪膛”，并对升级池/商店加入武器特有加点隔离
- 新增敌对霰弹怪 `shotgunner`：近距离多弹丸、低速、会逐渐消失
- `侧翼弹夹`（原多侧射逻辑）削弱为每次仅增加 1 条额外弹道
- 武器选择页改为纵向滚动式列表，支持滚轮、↑↓ 与数字键，解决新增武器后的拥挤重叠
- 霰弹枪子弹加入更强击退；左上角 HUD 重做为分区面板样式，信息层次更清晰
- 废料堆场新增绿色特色怪 `toxic_bloater`（死亡释放毒雾），反应堆室新增蓝色特色怪 `reactor_bomber`（死亡爆炸）
- 2x2 / 2x3 / 3x3 房间分割墙变薄，并对墙段做额外延伸，避免缩窄后出现漏缝
