# 钢铁蜂巢

这是一个基于 `pygame` 的俯视角门切房肉鸽原型工程，项目根目录位于 `T:/gameV1`。

## 当前玩法

- `WASD` 移动角色
- 鼠标左键射击
- `Space` 冲刺，`Q` 释放脉冲
- 已接通 **玩家 / 敌人 / 子弹 / 掉落 / 升级** 的完整循环
- **房间地图系统**：
  - 通过北 / 东 / 南 / 西门切换相邻房间
  - 包含战斗、精英、商店、宝箱、Boss 等房型
  - 右上角显示当前楼层小地图
  - 已支持大房、2x2、2x3、迷宫等布局
- 房间主题已包含开阔车间、掩体工带、废料堆场、反应堆室、封锁壁垒
- 支持可破坏障碍、毒气障碍、爆炸障碍、箱墙与主题障碍组合
- 已加入升级三选一、商店强化、晶片货币、补给房、纪录系统
- Boss 传送门现在需要二次确认，并带简短下层转场
- Boss 新增近距离撼地与中距离震荡齐射两种特殊攻击
- 新增可选武器 **废土霰弹枪**，并加入敌对霰弹怪
- 主菜单已拆分为关卡 / 角色 / 武器独立子页面
- 武器页已改为纵向滚动式列表，适配更多武器选项
- 左上角 HUD 已重做为分区状态面板
- 废料堆场 / 反应堆室加入主题特色怪物

## 2026-04-16 平衡更新

- 升级收益改为更平滑的小步成长，避免中期数值膨胀过快
- 敌人生命、接触伤害与远程伤害会随关卡难度和楼层继续成长
- 商店价格会随楼层和房间难度提高，后期决策更重要
- 新增更多成长选项：
  - 降低敌方子弹速度
  - 提高晶片获取倍率
  - 子弹获得一次反弹
  - 扩大脉冲范围
- 普通枪反射上限保持 1 次，激光武器可叠加反射至 3 次
- 爆炸桶伤害提高 50%，机枪怪（`shooter`）攻击频率下调
- 怪物寻路改为独立导航模块，能更稳定地绕过障碍与拐角
- `elite` 重做为高攻低防的 burst 威胁单位：血量较低，但会间歇性连续开火
- 新增 `shotgunner` 霰弹怪：近距离喷出低速弹丸，弹丸会逐渐缩小消失
- 升级池支持按武器隔离；霰弹枪独有“加长枪膛”不会出现在其他武器上
- `侧翼弹夹` 现在每次只追加 **1 条**额外弹道，不再一次给两侧各一条
- 抽取升级时会尽量规避已经到上限的选项，减少“废选项”
- 将内容配置与数值缩放从 `game.py` 中拆出，便于后续继续扩展
- 霰弹枪的击退感增强；房间分割墙体变薄并延长，减少视觉与寻路缝隙

## 代码结构

```text
T:/gameV1
├─ main.py
├─ README.md
├─ CHANGELOG.md
└─ game/
   ├─ game.py          # 主循环、战斗、房间切换、UI
   ├─ map_system.py    # 地图生成、布局拼接、房间模板
   ├─ entities.py      # Bullet / Enemy / Pickup 等基础实体
   ├─ config.py        # 全局常量与颜色
   ├─ content.py       # 升级、角色、武器、补给与标题页文案
   ├─ balance.py       # 敌人缩放、商店价格、奖励掉落等平衡公式
   └─ navigation.py    # 房间内导航网格 / Flow Field 寻路
```

## 操作说明

- `WASD`：移动
- 鼠标左键：射击
- `Space`：冲刺
- `Q`：脉冲
- `E`：交互（商店 / 宝箱 / 传送门确认）
- `ESC`：暂停 / 继续
- `R`：重开当前局
- `M`：返回标题菜单

## 新增武器与敌人说明

- **废土霰弹枪**
  - 一次发射多枚近距离弹丸
  - 独有升级：**加长枪膛**（增加飞行距离，并略微收束扩散）
- **Elite**
  - 高攻低防，属于优先击杀目标
  - 会在短暂停顿后进行一小段连续 burst 射击
- **Shotgunner**
  - 会主动贴近后喷出一片低速霰弹
  - 弹丸会随飞行时间逐渐缩小并消失
- **Toxic Bloater**
  - 出现在废料堆场
  - 死亡后会释放持续毒雾
- **Reactor Bomber**
  - 出现在反应堆室
  - 死亡后会触发蓝色爆炸

## 开发建议

- 每次修改后先执行 `python -m py_compile`
- 使用 dummy SDL 做基础烟雾测试，优先确保标题页、子菜单和开局流程不报错
- 若发现中文显示异常，优先检查源文件编码与字符串是否污染
- 后续新增内容建议优先放进 `content.py` / `balance.py` / `navigation.py`，避免继续堆积到 `game.py`
