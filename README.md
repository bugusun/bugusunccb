# 钢铁蜂巢

这是一个基于 `pygame` 的俯视角门切房肉鸽原型工程，项目根目录位于 `T:/gameV1`。

## 当前玩法

- 使用 `WASD` 移动、鼠标左键射击、`Space` 冲刺、`Q` 释放角色技能
- 标题菜单已拆分为 **关卡 / 角色 / 武器** 独立页面，可在开局前分别选择：
  - 起始关卡：第一关 / 第二关 / 第三关（分别从第 1 / 3 / 6 层开始）
  - 角色：先锋机体 / 曼巴奥特 / 工蜂机体 / 坤坤
  - 武器：制式步枪 / 蜂群机枪 / 废土霰弹枪 / 猎隼狙击枪 / 铁锤火箭炮 / 脉冲激光 / 棱镜重激光
- 已接通 **玩家 / 敌人 / 子弹 / 掉落 / 升级 / 商店 / 宝箱 / 纪录** 的完整循环
- **房间地图系统**：
  - 通过北 / 东 / 南 / 西门切换相邻房间
  - 包含战斗、迷宫、精英、商店、宝箱、Boss 等房型
  - 每 3 层有概率将 Boss 房升级为 **高难首领房**
  - 右上角显示当前楼层小地图
  - 已支持 1x1、1x2、1x3、2x2、2x3、中央环墙与 3x3 迷宫等布局
- 房间主题包含开阔车间、掩体工带、废料堆场、反应堆室、封锁壁垒，并已接通主题敌人：
  - 开阔车间：`engineer`
  - 掩体工带：`turret`
  - 废料堆场：`toxic_bloater`
  - 反应堆室：`reactor_bomber`
- 支持可破坏障碍、毒气障碍、爆炸障碍、反应堆障碍、箱墙与主题障碍组合
- 单厅 / 环墙房现在还会随机出现 **核弹事件** 或 **精英炮塔事件**
  - 核弹被打爆后会造成大范围爆炸、生成毒云，并给玩家附加 **辐射**（期间无法回血，攻击会附带中毒）
- 已加入升级三选一、宝箱三选一、商店强化、晶片货币与纪录系统
  - 商店为 **单房限购 2 次**
  - 预置补给选项当前包含：**应急维修 / 火力过载 / 电容充能**
- Boss 传送门需要二次确认，并带简短下层转场
- 普通 Boss 已有二阶段，并带专属 Boss 血条
- 高难 Boss **歼灭主宰** 额外带有冲刺压制与召唤机制
- 左上角 HUD 已重做为分区状态面板；奖励 / 补给 / 暂停 / 死亡面板也已统一风格

## 操作说明

- `WASD`：移动
- 鼠标左键：射击
- 鼠标右键：对 **猎隼狙击枪 / 铁锤火箭炮 / 棱镜重激光** 启用自动瞄准锁定
- `Space`：冲刺
- `Q`：释放当前角色技能
- 长按 `Q`：
  - **先锋机体**：消耗 100 晶片释放每层一次的全屏震撼
  - **坤坤**：消耗 500 晶片释放每层一次的八方向晶片齐射
- `E`：交互（商店 / 宝箱 / 传送门确认）
- `1 / 2 / 3`：在升级、宝箱奖励等选择界面中快速选择
- `Enter`：确认下潜到下一层
- `ESC`：暂停 / 继续，或在下潜确认框中取消
- `R`：重开当前局
- `M`：返回标题菜单

## 角色 / 武器 / 房间 / 敌人说明

### 角色

- **先锋机体**
  - 更厚实的前线框架，生命与护盾更高
  - 技能：**电弧脉冲**，推开近身敌人并清除周围敌弹
  - 长按 `Q`：消耗 100 晶片释放每层一次的 **全屏震撼**，可清弹并群体震晕

- **曼巴奥特**
  - 黄棕突进机体，上棕下黄，游戏内绘制带有 `24` 号标识
  - 技能：**曼巴重击**，前摇后向前方区域挥出重击，造成伤害、强击飞与眩晕
  - 被动：首次死亡会原地复活，并获得短时间霸体
  - 专属升级：**what can i say**（增强曼巴重击）

- **工蜂机体**
  - 偏技能与资源调度的支援框架
  - 被动：脉冲恢复更快，开局自带晶片

- **坤坤**
  - 均衡型表演机体
  - 被动：每持有 1000 晶片伤害 +10%，上限 +30%
  - 技能：**弹射篮球**，放出会无限反弹、清房后消失的篮球
  - 长按 `Q`：消耗 500 晶片释放每层一次的 **晶片齐射**
  - 专属升级：**篮球实习生**（增大篮球体积并提高速度）

### 武器

- **制式步枪**：中速、中伤、泛用性最好
- **蜂群机枪**：高射速压制，但单发更轻、扩散更大
- **废土霰弹枪**：近距爆发，多弹丸散射，独有升级 **加长枪膛**
- **猎隼狙击枪**：高伤高准，侧弹道会成对增加，支持右键自动瞄准
- **铁锤火箭炮**：低频重火力，命中后爆炸并击飞，可用 **广域装药** 扩大爆炸范围，支持右键自动瞄准
- **脉冲激光**：低伤高频的贯穿直线火力
- **棱镜重激光**：高伤低频的聚焦激光，带明显后坐力，支持右键自动瞄准

### 房间 / 事件

- **迷宫房**
  - 带独立迷宫墙体与可破坏墙段
  - 未清房时会保留来路撤退门，前进门保持锁定

- **宝箱房**
  - 开箱后进入三选一强化界面

- **商店房**
  - 房内限购 2 次
  - 商品价格会随楼层与房间难度提高

- **预置补给选项**
  - 当前数据包含 **应急维修 / 火力过载 / 电容充能** 三项

- **随机房间事件**
  - **核弹事件**：打爆后会造成核爆、毒云和辐射
  - **精英炮塔事件**：会额外生成更强的炮塔威胁

- **Boss 房**
  - 击败后激活传送门
  - 进入下一层前需要二次确认
  - 普通 Boss 会在半血后进入二阶段
  - 每 3 层有概率改为 **高难首领房**，Boss 变为 **歼灭主宰**

### 敌人

- **Elite**
  - 高攻低防，属于优先击杀目标
  - 会在短暂停顿后进行一小段连续 burst 射击

- **Shotgunner**
  - 会主动贴近后喷出一片低速霰弹
  - 弹丸会随飞行时间逐渐缩小并消失

- **Engineer**
  - 开阔车间主题敌人
  - 对护盾更具威胁

- **Turret / Elite Turret**
  - 掩体工带与事件房的定点火力单位

- **Toxic Bloater**
  - 出现在废料堆场
  - 死亡后会释放持续毒雾

- **Reactor Bomber**
  - 出现在反应堆室
  - 死亡后会触发蓝色爆炸

## 开发建议

- 每次修改后先执行 `python -m py_compile`
- 使用 dummy SDL 做基础 smoke test，优先确保标题页、子菜单和开局流程不报错
- 若发现中文显示异常，优先检查源文件编码与字符串是否污染
- 后续新增内容建议优先放进 `content.py` / `balance.py` / `navigation.py`，避免继续堆积到 `game.py`
